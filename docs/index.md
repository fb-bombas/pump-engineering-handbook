# Pump Engineering Handbook

Welcome. This handbook collects what we know about industrial pump
engineering — written for engineers specifying equipment, buyers comparing
options, and operators commissioning systems.

It is published openly by [FB Bombas](https://www.fbbombas.com.br), a
Brazilian pump manufacturer operating since 1944.

## Sections

1. **Centrifugal pumps** — normalized geometry, selection, hydraulic testing
2. **Gear pumps** — external and internal, viscous fluid handling
3. **Thermal-oil pumps** — closed-loop heating circuits
4. **Fire-fighting pumps** — NFPA-20 and NBR-16704 compliance
5. **Glossary** — trilingual (PT / EN / ES) terminology

## How we use real models as examples

When a discussion benefits from a concrete example, we cite a model from
our own catalog (FBCN, FBE, FBEI, FBOT). The math itself is generic and
holds for any manufacturer's equipment with the same operating envelope.

If you spot an error, [open an issue](https://github.com/fb-bombas/pump-engineering-handbook/issues).
